🟡 Discogs — Live Sync

Discogs API works directly from the browser using a personal token. Orders, prices, and marketplace fees are pulled automatically.

Get your token at discogs.com/settings/developers → Generate new token.
Your token is stored locally in your browser only.

🔴 eBay — API Notes

eBay requires OAuth 2.0 which can't be done securely from a plain browser file (it exposes your client secret). To sync eBay live data you need a small backend proxy.

Recommended approach: Export your eBay sales as a CSV from Seller Hub → Reports → Downloads and use the CSV Import tab below to bulk-load your history.

API endpoints for a future backend:
• Sales: GET /sell/fulfillment/v1/order
• Fees: GET /sell/finances/v1/transaction

⚙ Fee Rate Settings

These rates are used by the auto fee calculator in the Add Item form. Adjust to match your seller tier.
